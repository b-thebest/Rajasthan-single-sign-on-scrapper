To run this bot you need to have following:
1. Python 3
2. pycharm editor 

Required packages need to be installed in your system to run this code:
1. selenium
2. pandas

To install python simply download python(3.6) from python.org and install it 
Install pycharm community from its official website 

To install required packages install pip in your system using cmd:

--> sudo apt install python3-pip or apt install python3-pip
--> pip3 install pandas
--> pip3 install selenium 

To run the auto bot do the following operations :
--> Insert details in excel file named "person_details.xlsx" [NOTE : NEVER CHANGE NAME OF FILE OR HEADERS OF EXCEL FILE, ONLY EDIT DETAILS  OF  MEMBERS]
--> Open this project in pycharm editor and run the file rjbot.py
--> Bot will fill all the details and login and you just need to select date zone, place and vehicle after that system will fill all the information on its own [Don't Interrupt it then]. 
--> After details get filled enter captcha and submit details.
--> To change the password and username of a user you need to change it from a python script
